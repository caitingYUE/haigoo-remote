# Haigoo Remote Assistant

## 简介
Haigoo Remote Assistant 是一个专注于远程工作的招聘与求职平台助手。

## 部署状态
[![Vercel Status](https://deploy-badge.vercel.app/project/haigoo-remote)](https://vercel.com/dashboard)

Last Deployed: 2025-12-10 (Forced Update)

## 功能特性
- 职位聚合与筛选
- 简历解析与匹配
- 可信企业管理
- RSS 订阅源管理
- 自动化标签分类

## 开发
```bash
npm install
npm run dev
```

## 部署
本项目部署在 Vercel 上。
