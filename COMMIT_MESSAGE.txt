feat: 实现简历解析功能（前端+后端双方案）

## 问题
- 本地环境上传简历文件后解析失败（TIKA_URL 未配置）
- Vercel 正式环境无法正常上传简历（Edge Function 配置问题）

## 解决方案
实现了前端优先、后端备用的双轨制解析方案：

### 1. 纯前端解析器（主力方案）
- 使用 PDF.js (CDN) 解析 PDF
- 使用 JSZip 解析 DOCX
- 原生 API 处理 TXT
- 速度快（1-3秒）、零后端消耗、自动回退

### 2. 轻量级后端解析器（备用方案）
- 使用 pdf-parse、mammoth 等轻量库
- Vercel Serverless Function
- 统一解析逻辑、可扩展性强
