# 小程序上线测试清单

## 身份矩阵

- [ ] 未登录访客可以浏览，不能收藏或申请。
- [ ] 微信已登录但未绑定账号时，收藏/申请会引导绑定。
- [ ] 已绑定免费用户可以同步收藏和申请入口记录。
- [ ] 未验证邮箱用户能收到明确提示。
- [ ] 有效会员无订阅时能创建订阅并同步网站与邮箱。
- [ ] 有效会员已有网站订阅时能直接看到订阅与岗位更新。
- [ ] 过期会员回到会员服务展示，不再访问会员订阅能力。

## 岗位与同步

- [ ] 列表加载第 2、3 页，不停在 20 条。
- [ ] 默认排序与网站一致，最新排序按时间倒序。
- [ ] 七个分类组合分页无重复、无跳页。
- [ ] 搜索、列表曝光和详情累计 100 个唯一岗位，同一岗位不重复计数。
- [ ] 收藏在小程序和网站双向新增/取消且重复请求无副作用。
- [ ] 申请入口复制后显示“已打开入口”，用户确认后显示“已申请”。
- [ ] 官网、邮箱、内推三种入口均可复制有效内容。
- [ ] 企业信息 Tab 正常展示，缺失字段使用空状态且不显示假数据。
- [ ] 订阅保存后网站、小程序和邮件任务使用同一订阅记录。

## 账号、安全与异常

- [ ] 注册必须勾选协议，数据库存在协议版本与时间。
- [ ] 连续错误绑定第 6 次返回 429；注册/重置第 4 次返回 429。
- [ ] 找回密码不泄露邮箱是否存在。
- [ ] 退出只清除本机状态；解绑保留网站账号；注销删除账号并设置 30 天锁。
- [ ] 弱网、超时、CloudRun/Vercel 故障有可理解提示且不会重复扣次数。
- [ ] iOS、Android、字体放大、全面屏安全区、长标题、空列表和 Logo 失败均通过。
- [ ] 新版本下载完成后出现重启更新提示。

## 审核材料

- [ ] 小程序 ICP 备案状态正常。
- [ ] 服务类目、主体名称、协议主体、客服电话/邮箱一致。
- [ ] 微信后台隐私保护指引已发布。
- [ ] 审核账号与操作路径已写入审核备注。
- [ ] CloudRun 公网访问关闭，默认测试域名未写入客户端。
- [ ] 生产监控、余额告警、自动续费、备份和回滚镜像已确认。
