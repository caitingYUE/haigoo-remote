# 一、先从真实用户体验角度，分析你当前页面的问题

我模拟一个“从国内找远程工作、希望找靠谱公司”的用户进入该页面时的感受 ↓

---

## 1. 我不知道「可信企业列表」能给我带来什么价值

用户看到标题：

> 可信企业列表

但：

* 缺乏说明这份列表为什么可信
* 没说经过怎样的筛选
* 没展示“我们和哪些公司有联系/合作/内推可能”

用户不知道这页能帮他什么。

建议：
必须加入价值说明，如：

> 我们根据远程文化、全球化友好度、招聘透明度、岗位真实性、对中国候选人的友好程度，挑选出的“可信远程优先企业库”。

---

## 2. 缺少“国内可申友好度”提示（你产品的核心差异化）

很多用户会问：

* 这个公司是否接受中国申请人？
* 是否强制美区时区？
* 是否需要母语英语？
* 是否对中国护照友好？

建议：
每个公司卡片都应该明确：

✔ 允许中国居住地申请
✔ 不限制国籍
✔ 无时区要求／需部分重叠
✔ 有中国员工
✔ Haigoo 企业认可级别（如 Gold / Silver）
✔ 是否提供内推

这些标签是你最独特的价值。

---

## 3. 页面视觉层级不够清晰，信息密度较高

当前列表呈现如下：

公司封面 → Verified → 公司名 → 描述 → Remote First → Open Positions

这个信息顺序对用户不是最重要。

用户最关心的其实是：

* 公司实力与远程文化可信度
* 是否接受中国人
* 是否支持内推
* 是否有活跃岗位
* 公司提供什么类型岗位

建议重新排序：

1. 公司名 +“可信级别”标识
2. 是否可国内申请
3. 远程文化（Remote First/Hybrid）
4. 公司亮点（行业、规模、融资）
5. 公司能提供的岗位类型
6. 当前活跃岗位
7. 内推可用性（如果有）

---

## 4. 卡片内容没有统一规范（部分有描述、部分没有）

用户浏览时会觉得页面不够专业。

建议统一结构：

* 品牌封面图
* Logo
* 公司名称
* 可信等级（如你真有人工精选，这很重要）
* 公司描述（统一字数，例如两行）
* 远程文化标签
* 中国友好度标签
* 岗位数量（超过多少时显示“岗位多，推荐关注”）
* 一个 CTA：查看公司详情 / 查看岗位

---

## 5. 缺少“公司详情页”

可信企业列表非常适合做一个：

> 公司详情页（类似 Apple / YC Directory 那种专业卡片）

内容可以包括：

* 公司介绍
* 招聘风格
* 工作时间要求
* 团队规模分布
* 是否接受中国时区、国籍
* 薪资透明度
* 在 Haigoo 上架的岗位
* 内推按钮入会引导（核心转化点）

你现在缺这一步。

# 二、单个页面（企业列表页）的结构优化建议

### 建议结构（可直接用于设计）

```
[页面标题]
可信远程企业精选榜

以及是否对中国地区开放等关键指标评估筛选。

[筛选区]

- 地区
- 是否支持国内申请（默认选中）

[展示区：企业卡片]
----------------------------------
[封面图]
[Logo + 公司名]
[可信级别：Gold/Silver/Bronze 或 Verified]

- Remote First

- 行业标签（SaaS / AI / E-commerce）

[Open Positions]
显示 3 个岗位（若多可点击展开）
或显示 “暂无岗位，但值得关注”

[按钮]
查看企业详情 →
----------------------------------
```

# 三、从网站整体交互角度，这个页面应该扮演什么角色？

这是关键点，你的可信企业列表不只是浏览页面，它应该：

## 1. 成为“打造信任”的关键入口

对于远程求职，用户最大痛点是：

* 岗位真假？
* 公司会不会骗简历？
* 会不会是虚假招聘？
* 是否愿意雇中国人？

这个页面应该负责解决上述所有心理阻力。

**建议定位：可信背书目录 + 入会引导**

---

## 2. 成为“提升 Haigoo 品牌高度”的 Showcase 页面

如果你未来要做：

* 海外远程人才平台
* 企业合作
* 中国用户的远程社区

可信企业列表是展示行业资源的最佳地方。

建议加入：

* 合作企业数
* 已服务求职者数
* 内推成功案例
* 你们的筛选标准公开透明（增加品牌力度）

---

## 3. 要服务列表页的筛选逻辑（与岗位页联动）

你可以让用户点击某个公司后：

* 列出该公司所有岗位
* 展示岗位是否支持国内申请
* 提供“关注该公司”功能
* 订阅新岗位通知

这些都是高转化动作。

---

## 4. 可加入“收藏企业”与“订阅企业动态”

用户与岗位并不总是匹配，但与公司文化可能匹配。

建议：

* 收藏公司
* 订阅“该公司更新了岗位”邮件
* 企业动态（新闻、融资、产品发布）

---

# 四、从交互方式讲：可信企业该用什么呈现方式？

## 推荐三种方式：

---

## 方案 1：Pinterest 卡片瀑布流（主打轻浏览）

适合用户快速扫公司名和感兴趣的品牌。

特点：

* 视觉丰富、高品牌感
* 模仿 Dribbble、Behance 的阅读体验
* 适合大量公司展示

---

## 方案 2：专业企业档案（类似 YC Startup Directory）

每一条企业都是专业档案，适合 B 端合作感。

特点：

* 严肃、专业、可信
* 格式统一
* 更符合“可信企业库”的定位

---

## 方案 3：可折叠的企业卡片（内容密度高但简洁）

点击卡片展开，查看岗位 + 企业详情。

适合：

* 你目前的数据结构
* 用户希望快速确认“是否值得投递”

---

# 五、最终推荐（适合你当前 Haigoo 产品定位）

我基于你的产品定位“国内求职者可申的海外远程岗位 + 精选企业合作”推荐：

## 最佳组合方案：

* 企业列表采用：**方案 2（专业企业档案型）**
* 企业详情采用：专业页（你已有岗位详情页风格）
* 列表页加入：筛选 + 排序（行业、国家、远程友好度）
* 企业卡片加入：

  * 是否可国内申请
  * 是否可内推
  * 当前岗位（最多 3 条 preview）
  * 公司亮点（AI、SaaS、全球团队规模）

同时，建议在：

### 岗位详情页 → 提供“来自可信企业”的标识

### 岗位列表页 → 添加“只看可信企业”过滤器

### 首页 → 添加“Haigoo 精选企业”区块，提高信任度

这会让整个站点的信任链路更完整。
